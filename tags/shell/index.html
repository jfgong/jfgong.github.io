更新照片的文件名

这次国庆回去,拍了一些照片,整理的时候,再次碰到纠结的问题,相机的默认命名方式是在独立目录下使用IMG_1234这种形式,当需要将不同目录下的照片汇总到一起的时候,就会出现重名的现象.这种命名方式几乎想不到有任何的意义,所以准备将之修改为相机中使用时间来命名的方式.

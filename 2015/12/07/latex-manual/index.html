使用latex

Latex是一种基于TEX的排版系统, 在一切皆文本的linux中, 使用latex可以方便的绘制图表和制作文档, 因为其丰富的宏包工具, 可以轻松处理复杂的表格, 绘图和数学公式等问题, 适合生成高质量的科学文档, 所以latex在科研写作中得到非常广泛的运用.

Install package

以安装pgfplots为例

  • 下载宏包并解压
  • 安装宏包
    • cp pgfplots.sty texmf/tex/latex/pgfplots/
    • 拷贝其他文件到texmf/tex下的对应目录
  • 更新索引, texhash或者mktexlsr
  • 检查是否安装成功, kpsewhich pgfplots.sty
    • 输出正确路径则为安装成功
    • 否则, 未安装成功
  • 使用, usepackage{pgfplots}

就像是安装软件会遇到依赖时一样, 如果新安装的宏包需要使用到其他宏包, 那么需要继续安装其依赖.

同样以pgfplots为例, 在画图时, 需要使用到tikz, 如果tikz未安装, 那么需要继续安装PGF&TikZ宏包.

Remove package

以上述tikz&pgf宏包为例:

  • 删除texmf\tex下的所有pgf文件
  • 更新索引, texhash或者mktexlsr
